INTERONET

a distributed system which contains the one master node called Global Domain Manager (gdm) and serveral slave node called Local Domain Manager (ldm).

The gdm is the master node which recieve the user request, allocate resources and call ldm. gdm is the independent of complicated type of devices in interonet.

The ldm is the slave node which dependent on serveral device (virtual-machine-manager, topology-transformer, power-system).
